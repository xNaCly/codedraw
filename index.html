Instructions

COLOR [color] Set a color, supports all values css would support (hex, rgba)
BOX [x,y,h,w] Draw a box, arguments are start (x,y) & height, width
WIDTH [width] Width for LINE
LINE [sx,sy,ex,ey] Draw a line from (sx,sy) to (ex,ey)
FONT [font] Font for TEXT ([size] [font])
